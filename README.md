# Deck Building

**PG29 Rapid Prototyping**

## About the Project

This project is to create three prototypes around deck building.


### What I did:

After getting feedback from my first prototype (not included) I decided to come up with 3 different prototypes. 

** Prototype 1 **

The first prototype was made in Unity. It was made with the idea of giving better user experience and adding card filtering. The main actions users did to create decks included being able to create multiple decks, add cards, remove them and filtering cards. 

This prototype included:
- Deck and card saving
- Main menu
- Creating decks
- Filtering card by name and cost

Controls:
- `Left click` cards to move cards to, from deck and press buttons

** Prototype 2 **

The second prototype was also made in Unity and based around customization of the deck. I wanted to focus specifically on being able to customize the deck, the cards of the deck as well as a bit of animation feedback. 

This prototype included:
- Customizing colour of card
- Animation during moving cards to the deck
- Able to select cards to view details and customize colour
- Can customize the card back of the deck
- Can rename the deck

Controls:
- `Left click` cards to bring up the card select screen
    - Left click colours to change the card/deck color
- `Right click` a card to move from/to deck

The third prototype was made in Vue. For this prototype I focused on card fusions and card trading.

This prototype included:
- Fusions between 2 cards to create new cards
    - System to have parent cards stats affect fused cards
- Card trading
    - Ability to decline / accept the new trade

More detailed notes including the empathy map is inside `dev log.pfd`

## How To Use Prototype 1 and 2

**Step 1** Open the prototype 1 folder.

**Step 2** Open `Prototype1.exe` to start playing

**Step 3** Repeat for prototype 2

## How To Use Prototype 3

**Step 1** Open the prototype 3 folder.

**Step 2** Type `cmd` to enter the command line

**Step 3** Type `npm install` to install all the packages

**Step 4** Type `npm run dev` to run the web app
